# Minesweeper (Tkinter GUI)

A clean Minesweeper clone built in Python using Tkinter.  
Includes difficulty levels (1–5), first-click safety, timer, flag counter, border + inner panel styling, and dark mode.

## Features
-  Tkinter GUI
-  Difficulty levels 1–5 (wider boards)
-  First click is always safe (mines placed after first reveal)
-  Right-click to flag
-  Timer + flag counter
-  Dark mode toggle
-  Bordered play area with inner background panel

## Controls
- **Left click**: Reveal cell  
- **Right click**: Toggle flag  
- **New Game**: Restart using selected difficulty  
- **Dark mode**: Toggle UI theme  

## Requirements
- Python 3.10+ (works on most Python 3 versions)
- Tkinter (usually included with Python)

## Run
```bash
python minesweeper.py
